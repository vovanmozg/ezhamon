Ежамон
========
Адрес проекта: http://vovanmozg.com/ezhamon/
Автор: Вован Полухин (http://vovanmozg.com)


Установка Ежамона
-------- 
1. Распакуйте содержимое архива на свой сервер. У вас должна появиться каталог ezhamon.
2. Установите права 777 на каталог ezhamon. Это можно сделать из FTP-менеджера или из файлового менеджера, если он
   есть на вашем хостинге.
3. Зайдите с помощью браузера по адресу http://ВашСайт/ПутьКЕжамону/
   Например, если папку со скриптами вы разместили в корневом каталоге сайта mysite.ru, то вам нужно перейти по адресу
   http://mysite.ru/ezhamon/
4. Следуйте инструкциям по установке. Инсталлятор должен создать все нужные таблицы.

Вместо пунктов 3 и 4 процедуры установки можно просто внести в файл config.php сведения о базе данных.

5. Удалите файл install.php
6. Для того чтобы парамтеры сайтов обновлялись автоматически нужно настроить периодическое выполнение скрипта refresh.php
   с помощью cron.
	

Работа с Ежамоном
-------- 
Особо расписывать нечего.
Добавить сайт можно на странице "Управление".
На главной странице отображается страница со список сайтов и их параметрами.
Чтобы обновить сео-параметры очередного сайта из списка нужно обратиться к скрипту refresh.php.
Чтобы настроить систему на автоматическое обновление, нужно настроить периодический запуск refresh.php с помощью cron.
Можно выставить интервал запуска 10 минут.

Особенности работы
--------
При каждом запуске скрипт получает параметры одного и только одного сайта, причём только того, с момента последнего обновления которого прошло не менее суток. Это сделано для того, чтобы один и тот же сайт не проверялся по несколько раз в сутки, и в то же время чтобы за сутки проверились все сайты из списка.

Если посчитать, получается что система может проверять максимум 86400 сайтов в сутки (60*60*24).
